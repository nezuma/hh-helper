## 📋 О проекте

**HH Helper** — это десктопное приложение, которое объединяет два крупнейших сервиса по поиску IT-работы в России: **hh.ru** и **Хабр Карьеру**. Больше не нужно переключаться между вкладками браузера — все ваши отклики, приглашения и сообщения в одном месте.

### 🎯 Основная идея

Мы не автоматизируем процесс поиска работы, а делаем его максимально удобным и централизованным. Вы сами контролируете процесс, но тратите на него значительно меньше времени.

---

## ✨ Возможности

### 🎯 Централизация откликов

Все отклики с hh.ru и Хабр Карьеры в одном месте. Больше не нужно переключаться между сайтами и терять важные сообщения.

### 📊 Детальная статистика

Анализируйте эффективность поиска:

- Количество отправленных откликов
- Приглашения на собеседования
- Успешные наймы
- Процент отказов

### ⚡ Мгновенные уведомления

Получайте уведомления о:

- Новых вакансиях по вашим критериям
- Приглашениях на собеседования
- Изменениях статуса откликов
- Новых сообщениях от работодателей

### 📱 Кроссплатформенность

Работает на всех основных операционных системах:

- Windows 10 и выше
- macOS 11 и выше
- Linux (Ubuntu 20.04+, Fedora 35+, Debian 11+)

### 🔒 Безопасность данных

- **Никаких паролей** — мы не храним ваши логины и пароли от сервисов
- **Только cookies** — для авторизации используются только сессионные cookies
- **Локальное хранение** — все данные хранятся на вашем устройстве
- **Прозрачность** — вы всегда можете посмотреть, какие данные сохранены

### 🎨 Удобный интерфейс

- Интуитивно понятный дизайн
- Темная тема для комфортной работы
- Адаптивный интерфейс для разных разрешений экрана

## 🚀 Установка

### Системные требования

| Компонент    | Минимальные требования                        |
| ------------ | --------------------------------------------- |
| **ОС**       | Windows 10 / macOS 11 / Linux (Ubuntu 20.04+) |
| **ОЗУ**      | 512 MB                                        |
| **Диск**     | 200 MB свободного места                       |
| **Интернет** | Стабильное подключение                        |

### Способы установки

#### Windows

```bash
# Скачайте установщик .exe с релизов
# Запустите HH-Helper-Setup.exe
# Следуйте инструкциям установщика
```
